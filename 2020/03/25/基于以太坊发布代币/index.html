进入 https://remix.ethereum.org 初始进去只有两个菜单，选插件安装如上图所示插件Deploy & run transactions和Solidity compiler

新建合约文件复制如下代码，不做任何修改

选择对应的版本开始编译

选择环境，这里选的是 Injected Web3 ，会根据你的MetaMask钱包网络显示你的钱包地址及余额，下面部署可以填参数，然后部署。不知道怎么填的可以点右边的箭头下拉出来
